Landsat-util

Landsat-util is a command-line tool created by Development Seed to enable simple and rapid searching, downloading, and processing of Landsat imagery. Whether you're a remote sensing expert or a newbie, landsat-util will speed up your workflow considerably.

This is a guide to getting started with landsat-util. For the full documentation, head over to Github.

Install

Before we begin, we need to install landsat-util.

On OSX use homebrew:

brew install https://raw.githubusercontent.com/developmentseed/landsat-util/master/Formula/landsat-util.rb

On Ubuntu, use pip:

sudo apt-add-repository ppa:ubuntugis/ubuntugis-unstable
sudo apt-get update
sudo apt-get install git python-pip build-essential libssl-dev libffi-dev python-dev python-gdal libgdal1-dev gdal-bin imagemagick geotiff-bin -y
sudo pip install -U git+git://github.com/developmentseed/landsat-util.git

If you're on Windows, or want to build from source, see the documentation.

Locate

There are a number of ways to use landsat-util to download imagery, but it's easiest if you know the path and row of the location you're interested in.

Use the map to identify the path and row of the location you're interested in.

For this guide, we're going to be using the scenes available at Path 22 - Row 39, which includes New Orleans.

Search

The first thing we'll use landsat-util for is to search through the available imagery for the scenes that match our criteria. In addition to providing our path and row, we can also filter by start and end date, and maximum percent cloud cover.

landsat search --start 'january 1 2014' --end 'july 1 2014' pr 022 039

Running that will provide you with JSON that includes two important keys to take note of: thumbnail and sceneID. Preview each of the thumbnails to conform that the results are appropriate, and then make not of the sceneID's that you'd like to download.

We're going to use LC80220392014099LGN00 and LC80220392014019LGN00 for the next step.

Download

Next, let's download the scenes.

landsat download LC80220392014019LGN00
landsat download LC80220392014099LGN00

Scenes will be downloaded to your ~/landsat directory. If the directory doesn't exist, it will be created for you. Each download will be around 700mb, so go take a walk and make some coffee.

Process

Our download has finished, and we have our data. If you unzip the scenes, you'll notice that rather than having a single image, we have 11. Each of those images contains a particular range of frequencies along the electromagnetic spectrum. While they may be useful alone, the real power of landsat imagery lies in combining those bands together.

Landsat-util makes it easy to automate the process of generating true-color images; let's go ahead and do that.

landsat process --pansharpen ~/landsat/zip/LC80220392014099LGN00.tar.bz
landsat process --pansharpen ~/landsat/zip/LC80220392014019LGN00.tar.bz

Once processed, we can do all sorts of amazing things with the data. Here I've followed Mapbox's instructions on using Tilemill to generate map tiles.
